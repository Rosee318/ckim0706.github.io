  Hash Index    hash[key]
          (default value if out of bounds)

# Iterators

  Each Key    .each_key
  Each Value    .each_value

  Select      .select

Array and Hash in Ruby helps programmers to efficiently manage data. 
Array and Hash both are able to store multiple types of data inluding boolean (t/f), numbers and text. 
The main difference between Array [ ] and Hash { } are the symbols to create and its flexibility. Hash { } needs a key and a value. 

Examples:
array = [true, 1, 2.0, "Hello", "bye"]
array.each { |x| puts "#{x}" }

hash = {tiger: "land", whale: "sea", eagle: "air"}
hash.each { |x, y| puts "#{x}: #{y}" }

As you can see .each on examples, there are numerous Enumerable classes that can be connected to array and hash. 
These enumerable classes such as .length, .size, .upcase , ETC. allow user to manipulate the output results.

Arrays

# Constructor

    array = Array.new   # => []
    array = Array.new(3)  # => [nil,nil,nil]
    array = Array.new(3, x) # => [x,x,x]
    
    copy = Array.new(array) # => [x,x,x]

# Operators

    Concatenate   +
    Append      <<
    Repeat      *
    
    Subtract    -
    Union     |
    Intersection  &
    
    Array Index   array[0]
    Subarray Index  array[3, length]
    Subarry Index array[3..5]
            (nil if out of bounds)
