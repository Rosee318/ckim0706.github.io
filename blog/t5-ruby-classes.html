Object-Oriented Ruby: Classes

October 17, 2014

Classes are objects that supports all the operations such as passing a parameter, returning functions and assigning a variable. Classes look like this: class Name

* hint* name of class must be capitalized

def initialize (first_name, last_name) @first_name = first_name @last_name = last_name end def greet puts "Hi, I'm #{@first_name} #{@last_name}" end end To create a new class and use the methods: me = Name.new("Charles","Kim")

* hint* parenthesis must be placed without white_space

me.greet ------ console "Hi, I'm Charles Kim" -------------------------------------- Classes have special variable called attr_. There are three types of attr_. attr_reader :first_name def first_name @first_name end attr_writer :last_name def last_name=(last_name) @last_name = last_name end att_accessor :first_name, :last_name Both There are so much information regarding classes, thus again here's some good links: Ruby Learning Pragmatic Programmer's Guide The Score - 3 Ways to Create Classes
