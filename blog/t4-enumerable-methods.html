Ruby Enumerable Method: Map

October 9, 2014

Map is one of the most used Enumerable or a "module", where it can modify the given object. There are two ways to use map. Map returns a NEW array which is returned by the block. Object.map { |item| block } → new_array Object.map → Enumerator Example: a = [1,2,3] a1 = a.map {|x| x*2} a1 = [2,4,6] Map! allows the same effect, where each of the blocks replace the original item. Example: a = [1,2,3] a.map {|x| x*2} a = [2,4,6] I found the following links to be helpful in my learning. Links: Erick Trautman's Blog Robert Qualls' Ruby Tips
