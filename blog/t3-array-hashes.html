Array and Hashes in Ruby

October 4, 2014

Array and Hash in Ruby helps programmers to efficiently manage data. Array and Hash both are able to store multiple types of data inluding boolean (t/f), numbers and text. The main difference between Array [ ] and Hash { } are the symbols to create and its flexibility. Hash { } needs a key and a value. Examples: array = [true, 1, 2.0, "Hello", "bye"] array.each { |x| puts "#{x}" } hash = {tiger: "land", whale: "sea", eagle: "air"} hash.each { |x, y| puts "#{x}: #{y}" } As you can see .each on examples, there are numerous Enumerable classes that can be connected to array and hash. These enumerable classes such as .length, .size, .upcase , ETC. allow user to manipulate the output results. I found this website to great resource to see different classes Link: Dev Docs
